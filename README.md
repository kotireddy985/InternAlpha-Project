# Online Banking System

A GUI-based banking application developed using Python, Tkinter, and SQLite.

## Features

- User Registration
- User Login
- Deposit Money
- Withdraw Money
- Check Balance
- Transaction History
- SQLite Database Storage

## Technologies Used

- Python
- Tkinter
- SQLite3

## How to Run

```bash
python gui.py
```
